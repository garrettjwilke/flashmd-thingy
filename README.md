# flashmd-thingy

## Requirements
- Qt 5 or Qt 6 with Core, Gui, Widgets, and SerialPort modules
- qmake (from Qt)
- make
- C++11-capable compiler (clang or gcc)

## How to build
- From the project root:

```
qmake qmakefile.pro
make
```

- The resulting binary will be created at:
- build/flashmd-thingy


## how to run

```
./build/flashmd-thingy
```
